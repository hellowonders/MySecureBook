MySecureBook Help

MySecureBook is a advance blockchain based documents storage solution which uses AES-128 bit encryption standard. We have used a newly invented concept 'Progressive Blockchain' in this software to prevent data loss due to invalidation of traditional blockchain when a block has been deleted/corrupted.

Any data saved cannot be modified by any means to maintain authenticity and reliability. If such tempering/hacking is being tried, the project will get invalidated and cannot be used further. However, previously stored data can be recovered.

Some Use Cases

1. User can store health receipts in a sequential manner over a period of time, the project can be shared to a doctor having full patient diagnosis history, it prevents need to carry physical prescriptions.

2. Business meeting notes can be saved using a timestamp to keep track of the whole project. Documents such as BRD, SOW etc. can be saved by versions.

3. Confidential personal information can be stored securely, AES-128 encryption is used by default.

4. Educational certificates can be maintained, they can be shared securely to a hiring company.

5. Keep track of expenses and bills by uploading invoices.

 

Software Usage

1. Enter license information to activate the product.

2. Create New Project -> Select Category -> Enter Project Name -> Click OK -> Create New Node -> Enter Details, select Attachment -> Click CREATE

Repeat the process of creating nodes to add more data. 

3. Explore Project -> Select Category -> Select Existing Project -> Open Project

Nodes can be added after opening project.

4. Download attachment by clicking on any node, it will turn green and information related to the seleted node will be shown, click Download button to retreive the original document. 

5. Export Project : Explore Project -> Select Category -> Export Project (.zip)

6. Import Project :  Explore Project -> Select Category -> Import Project (.zip)
